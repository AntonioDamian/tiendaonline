Remarks

Hay atributos que no pueden ser nulos,como usuarioId,email,nombre,apellidos,dni,puebloId y provinciaId, que son validados al intentar crear el usuario en la base de datos,en el constructor y las propiedades no son validados

Aunque podria admitir algun nulo en el constructor,sin estar registrado en la base no podria actuar,por lo tanto se valida al intentar agragarlo a la base de datos
